# LOVEFLIX (RomanceFlix+)

App de streaming de histórias de romance — protótipo visual do app + landing de venda.

## Acessar online

- **App (área de membros):** https://nardoto.github.io/LOVEFLIX/app.html
- **Landing de venda:** https://nardoto.github.io/LOVEFLIX/romance.html

## Arquivos

| Arquivo | O que é |
|---|---|
| `app.html` | Interface do app (área logada, estilo Netflix) |
| `romance.html` | Landing page pública pra captar assinantes |
| `biblico.html` | Versão alternativa com tema bíblico (preservada) |
| `capas/` | 45 capas cinematográficas (geradas com Imagen 3 / Nano Banana) |
| `PROMPTS-CAPAS.md` | Prompts numerados pra regenerar capas |
| `PASSO-A-PASSO-FINAL.md` | Cronograma técnico de 12 semanas |
